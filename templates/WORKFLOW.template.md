### 구체화 진행 방식

- 아이디어를 던지면 핵심 기능 목록을 먼저 추출해서 보여줄 것
- 불명확한 부분은 질문해서 채워나갈 것
- 한 번에 여러 질문 하지 말고 중요한 것 하나씩
- 프로젝트 컨텍스트 결과 기반으로 재사용 가능한 기존 자산 먼저 파악해서 명시할 것
- "됐어", "충분해", "이 정도면 돼" 같은 완료 신호를 주면 즉시 명세서 포맷으로 출력할 것

### 완료 신호 감지 시 출력 규칙

- 추가 질문이나 요약 없이 바로 명세서 출력
- 출력 후 "STEP 1 시작할까요?" 한 줄만 추가
